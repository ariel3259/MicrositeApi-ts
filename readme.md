# Microsites api ts
A reimagined api of MicrositesApi

## Enviroment

    . Nodejs 16.16.0
    . typescript 4.8.4
    . ts-node 10.9.1

## Dependencies

    . dotenv
    . express
    . express-validator
    . helmet
    . cors
    . sequelize

## Enviroment variables

    . PORT: server port
    . PORT_DB: database of port
    . USER_DB: username of database
    . PASSWORD: password of database
    . DATABASE: name of database
    . HOST: host of database
